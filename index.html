Improving Communication between Hospitals and Patients

Sendinel is a computer system, which aims at supporting clinics when they want to contact their patients. Via Sendinel the clinic staff can send SMS, a Voice Call or calendar entries to the patients’ mobile phones. For example, the messages can be reminders for follow–up consultations or notifications about arrived medicine. The development of Sendinel focused on clinics in rural South Africa.

Demo System

We have set-up a demo system which you can try out here.

The demo system provides the web frontend of the latest release, which is to be used by the staff in the hospital to control Sendinel. To avoid abuse the sending of SMS and Voice Call are disabled in the demo system.

The demo system is planned to be online at least until September 2010.

Installation

Setting up Sendinel is described in detail in the Installation Instructions which are available as html and pdf. The installation is aided by an installation script, which will install the needed packages.

Dependencies

  • Python 2.4 - 2.6
  • Django 1.2
  • Asterisk 1.6.2
  • Festival Speech Synthesis System 1.4

Requirements

Sendinel is best to be run under Linux. However it is also possible to run Sendinel on other platforms like Mac OS X or Windows when the environment described in Dependencies is set-up at your system.
