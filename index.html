About

With Sendinel you can send notifications and reminders via SMS and phone calls. Our main use case is the communication between clinics and patients. For more information please click here. You can also try out the frontend of Sendinel here.

Requirements

In order for Sendinel to run, you need to have a Huawei E169 3G data stick. (also sold as K3520)
To run Sendinel, Linux is recommended as operating system. However it is also possible to run Sendinel on other platforms like Mac OS X or Windows when the environment described in Dependencies is set-up at your system. We ran Sendinel successfully on the Linux distributions Ubuntu 9.10, 10.04 and Debian 6 (Squeeze).

Installation

Setting up Sendinel is described in detail in the Installation Instructions. Please read them (html|pdf).
In very short: After having downloaded the tar.gz file, you need to unpack it using "tar -xzf [filename]", change to that directory and run ./install.sh and everything should be fine!

Dependencies

On a compatible Linux distribution the installation script will resolve all dependencies automatically.
  • Python 2.5 - 2.6
  • Django 1.2
  • gcc 4.4 or newer
  • Asterisk 1.6.2 or newer
  • Festival Speech Synthesis System 1.4
